# LLM Calibration and Verbalized Confidence in Low-Resource Languages

> 🚧 Work in Progress

## Abstract
This research investigates how well large language models are calibrated
and how accurately they verbalize their own confidence in low-resource language settings.

## Repository Structure
- `data/` — dataset info and download instructions
- `src/` — source code and experiments
- `notebooks/` — EDA and analysis notebooks
- `results/` — figures, tables, and metric logs
