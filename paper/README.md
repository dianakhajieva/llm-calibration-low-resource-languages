# Paper

Put your LaTeX source (or a link to your Overleaf project) here.

Use the official ACL / ARR template, which is what UncertaiNLP requires:
https://github.com/acl-org/acl-style-files

Target: a 4-page short paper for the UncertaiNLP workshop at EMNLP 2026.
Submission deadline: August 7, 2026, via OpenReview.

Suggested section order:
1. Abstract
2. Introduction (motivation + your contributions)
3. Related work (LLM calibration, verbalized confidence, multilingual evaluation)
4. Method (data, models, the two confidence methods, the metrics)
5. Results
6. Analysis and discussion (with qualitative Uzbek examples)
7. Limitations
8. Conclusion
